# 第一章：决策内核 —— 物性向量与 PRA 路由
## 1. 物性向量（Property Vector）

不要用“好/坏”评价事物，而要用 10 个维度量化一个人、一个项目、一个决策：

- L（Logic 逻辑）  
- C（Connectivity 连通性）  
- E（Evidence 证据度）  
- V（Verifiability 可验证性）  
- R（Robustness 稳健性）  
- H（Humanity 人性）  
- U（Usability 好用度）  
- S（Scalability 可扩张性）  
- N（Novelty 新颖度）  
- T（Totality 闭环度）

物性向量 = 现实世界的“雷达图”。

> 高物性能量体自动吸引其他高物性能量体（密度匹配原则）。

---

## 2. PRA 路由算法（Pass / Refine / Abort）

这是龙一 OS 的决策引擎。

### ● PASS（通过）
条件：高物性向量 + 高正性预期  
动作：全力投入（All in）

### ● REFINE（修补）
条件：大方向对，但逻辑或新意不足  
动作：引入“遁一”，进行改写、头脑风暴、跳格思考

### ● ABORT（熔断）
条件：检测高黑子密度（欺骗、戾气、内耗）  
动作：立即切断，不纠缠，不解释

> PRA 是现实世界最稳定的进攻与防御框架。
