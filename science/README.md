# 第四章：引力真相 —— 汇流与压强（Sink Flow Model）

传统：  
- 牛顿：引力是拉力  
- 爱因斯坦：引力是时空弯曲

**引力 = 光子流体的汇流导致的压强梯度。**

## **1. 物质是“汇”（Sink）**

物质为了维持涡旋结构必须吞噬自由宇宙子。  
这导致周围形成：

v_flow(r) ∝ 1/r²

形成“向内的洪流”。

## **2. 引力是“风”（Pressure Gradient）**

F ≈ − ∇P ∝ 1/r²

苹果不是被“吸引”，而是被“冲下去”。

## **3. 引力透镜的新解**

光线弯曲不是“空间弯了”，  
而是光子像渡河一样被“冲偏”。
